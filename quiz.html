бһҳбҹҒбһҡбҹҖбһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—

бһўбһ¶бһ“бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ–бһ“бҹ’бһҷбһӣбҹӢбһӣбһҳбҹ’бһўбһ·бһҸ бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ”

Motherboard (бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҹбҹҖбһӮбҹ’бһңбһёбһҳбҹҒ)

Motherboard бһӮбһәбһҮбһ¶бһ”бһ“бҹ’бһ‘бҹҮбһҹбҹҖбһӮбҹ’бһңбһёбһ’бҹҶбһҮбһ¶бһ„бһӮбҹҒбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһҠбҹӮбһӣбһҠбһҫбһҡбһҸбһҪбһҮбһ¶ "бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһҡбһҹбҹғбһ”бҹ’бһҡбһҹбһ¶бһ‘бһҖбһҺбҹ’бһҸбһ¶бһӣ"бҹ” бһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбһјбһ…бһҮбһ¶ CPU, RAM, Hard Disk, VGA бһ“бһ·бһ„бһҡбһ“бҹ’бһ’бһҠбҹ„бһҸбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҹбһ»бһ‘бҹ’бһ’бһҸбҹӮбһҸбҹ’бһҡбһјбһңбһҠбҹ„бһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһңбһ¶бҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„ бһ“бһ·бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹ’бһ“бһ¶бһ‘бҹ…бһңбһ·бһүбһ‘бҹ…бһҳбһҖбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“бҹ”

CPU (бһҒбһҪбһҡбһҖбҹ’бһ”бһ¶бһӣбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ)

CPU (Central Processing Unit) бһ”бҹ’бһҡбҹҖбһ”бһ”бһ¶бһ“бһ‘бҹ…бһ“бһ№бһ„ "бһҒбһҪбһҡбһҖбҹ’бһ”бһ¶бһӣ" бһҡбһ”бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ” бһңбһ¶бһ‘бһ‘бһҪбһӣбһ”бһ“бҹ’бһ‘бһ»бһҖбһӮбһҺбһ“бһ¶ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҡбһ¶бһӣбҹӢбһ”бһ‘бһ”бһүбҹ’бһҮбһ¶бһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҸбһӣбҹӢбһІбҹ’бһҷбҹ” бһӣбҹ’бһ”бһҝбһ“бһӣбһҝбһ“ бһ¬бһҷбһәбһҸбһҡбһ”бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҳбһҪбһҷбһ—бһ¶бһӮбһ’бҹҶ бһӮбһәбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ‘бҹҶбһ бҹҶбһӣбҹ’бһ”бһҝбһ“ (GHz) бһ“бһ·бһ„бһ…бҹҶбһ“бһҪбһ“ Core бһҡбһ”бһҹбҹӢ CPU бһ“бҹҒбҹҮбһҜбһ„бҹ” бһҖбһ¶бһӣбһҺбһ¶ CPU бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһңбһ¶бһ“бһ№бһ„бһҮбһҪбһҷбһІбҹ’бһҷбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҠбһҫбһҡбһҹбҹ’бһҡбһҪбһӣ бһҳбһ·бһ“бһ„бһ¶бһҷбһӮбһ¶бҹҶбһ„бҹ”

RAM (бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“)

RAM (Random Access Memory) бһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷ CPU бһ‘бһ¶бһүбһҷбһҖбһ‘бҹ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“бһӣбһҝбһ“бһҮбһ¶бһ„бһ‘бһ¶бһүбһ–бһё Hard Disk бһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһҖбһ¶бһӣбһҺбһ¶бһҷбһҫбһ„бһҳбһ¶бһ“ RAM бһ’бҹҶ бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһўбһ¶бһ…бһ”бһҫбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ¬ Chrome Tabs бһ”бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбһҠбҹ„бһҷбһҳбһ·бһ“бһҷбһәбһҸбҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„ RAM бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһӣбһ»бһ”бһ…бҹ„бһӣбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһ–бҹҒбһӣбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһӣбһҸбҹӢбҹ”

GPU (бһҖбһ¶бһҸбһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһҖ)

GPU (Graphics Processing Unit) бһҮбһ¶бһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ“бҹ’бһ‘бһ»бһҖбһӮбһҺбһ“бһ¶ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бҹҒбһүбһҡбһјбһ”бһ—бһ¶бһ– бһңбһёбһҠбҹҒбһўбһј бһ¬бһ бҹ’бһӮбҹҒбһҳ 3D бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҡбһ…бһ“бһ¶бһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһҖ бһҖбһ¶бһҸбҹӢбһҸбһңбһёбһҠбҹҒбһўбһј бһ¬бһӣбҹҒбһ„бһ бҹ’бһӮбҹҒбһҳ бһҖбһ¶бһҸбһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһҖбһҠбһ¶бһ…бҹӢ (Dedicated VGA) бһӮбһәбһ–бһ·бһҸбһҮбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҒбҹ’бһӣбһ¶бҹҶбһ„бһҺбһ¶бһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһҡбһјбһ”бһ—бһ¶бһ–бһҠбһҫбһҡбһҡбһӣбһјбһ“бһӣбҹ’бһў бһҳбһ·бһ“бһҠбһ¶бһ…бҹӢбҹ— (Lag)бҹ”

Monitor (бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ)

бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ (Monitor) бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһү (Output Device) бһҠбҹҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ”бҹҶбһ•бһ»бһҸбһҳбһҪбһҷ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҡбһјбһ”бһ—бһ¶бһ– бһңбһёбһҠбҹҒбһўбһј бһ“бһ·бһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ…бҹҒбһүбһ–бһёбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһІбҹ’бһҷбһҷбһҫбһ„бһҳбһҫбһӣбһғбһҫбһүбҹ” бһӮбһ»бһҺбһ—бһ¶бһ–бһ“бҹғбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҖбһҳбҹ’бһҡбһ·бһҸбһ—бһ¶бһ–бһ…бҹ’бһ”бһ¶бһҹбҹӢ (Resolution), бһӣбҹ’бһ”бһҝбһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҡбһјбһ”бһ—бһ¶бһ– (Refresh Rate) бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бһ“бҹ’бһ‘бҹҮбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ (Panel Type) бһҠбһјбһ…бһҮбһ¶ IPS, VA бһ¬ TN бһҮбһ¶бһҠбһҫбһҳбҹ”

Keyboard & Mouse (бһҖбҹ’бһҸбһ¶бһҡбһ…бһ»бһ… бһ“бһ·бһ„бһҖбһҺбҹ’бһҸбһ»бһҡ)

бһҖбҹ’бһҸбһ¶бһҡбһ…бһ»бһ… (Keyboard) бһ“бһ·бһ„бһҖбһҺбҹ’бһҸбһ»бһҡ (Mouse) бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бһјбһӣ (Input Devices) бһ…бһҳбҹ’бһ”бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһҮбһ¶бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ” бһҖбҹ’бһҸбһ¶бһҡбһ…бһ»бһ…бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһңбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбҹ’бһҗбһ”бһ‘ бһ“бһ·бһ„бһҸбһҪбһўбһҖбҹ’бһҹбһҡбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҒбһҺбҹҲбһҠбҹӮбһӣбһҖбһҺбҹ’бһҸбһ»бһҡбһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһ…бһ„бҹ’бһўбһ»бһӣ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ бһ“бһ·бһ„бһўбһјбһҹбһ‘бһҳбҹ’бһӣбһ¶бһҖбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһӣбһҫбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮбһҮбһҪбһҷбһІбҹ’бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ”бҹ’бһҡбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ‘бҹ…бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһҡбһ бҹҗбһҹбҹ”

Storage (бһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ)

бһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Storage) бһҠбһјбһ…бһҮбһ¶ HDD (Hard Disk Drive) бһ“бһ·бһ„ SSD (Solid State Drive) бһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһўбһ…бһ·бһ“бҹ’бһҸбҹ’бһҡбҹғбһҷбҹҚбҹ” бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ SSD бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҒбҹ’бһӣбһ¶бҹҶбһ„бһҮбһ¶бһ„ HDD бһҠбҹ„бһҷбһҹбһ¶бһҡбһңбһ¶бһҳбһ¶бһ“бһӣбҹ’бһ”бһҝбһ“бһўбһ¶бһ“ бһ“бһ·бһ„бһҹбһҡбһҹбҹҒбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӣбһҝбһ“бһҮбһ¶бһ„бһҡбһ¶бһ”бҹӢбһҹбһ·бһ”бһҠбһ„ бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбҹ”

PSU (бһ”бҹ’бһҡбһ—бһ–бһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҗбһ¶бһҳбһ–бһӣ)

Power Supply Unit (PSU) бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһҷбҹүбһ¶бһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҷбһҖбһ…бһҡбһ“бҹ’бһҸбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбһ–бһёбһ–бҹ’бһҡбһёбһ—бҹ’бһӣбһҫбһ„ (AC) бһҳбһҖбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһ‘бҹ…бһҮбһ¶бһ…бһҡбһ“бҹ’бһҸ (DC) бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҗбһ¶бһҳбһ–бһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ” бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ PSU бһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһӣбҹ’бһў бһ“бһ·бһ„бһ‘бҹҶбһ бҹҶбһңбҹүбһ¶бһҸбҹӢ (Watt) бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢ бһӮбһәбһҮбһҪбһҷбһ’бһ¶бһ“бһ¶бһҠбһӣбҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһ¶бһҷбһ»бһҖбһ¶бһӣбһҡбһ”бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ”

Operating System (бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ)

бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ (OS) бһҠбһјбһ…бһҮбһ¶ Windows, macOS, бһ« Linux бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбҹҒбһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ•бҹ’бһ“бҹӮбһҖбһҡбһ№бһ„ (Hardware) бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸ (Software) бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ” бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“ OS бһ‘бҹҒ бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҳбһ·бһ“бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһЎбһҫбһҷ бһңбһ¶бһ”бҹ’бһҡбҹҖбһ”бһҠбһјбһ…бһҮбһ¶бһҡбһ¶бһ„бһҖбһ¶бһҷбһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“бһ–бҹ’бһҡбһӣбһ№бһ„бһўбһүбҹ’бһ…бһ№бһ„бҹ”

Application Software (бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ)

Application Software бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҺбһ¶бһҳбһҪбһҷбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– Microsoft Word бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ¶бһҷбһўбһҸбҹ’бһҗбһ”бһ‘, Google Chrome бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҒбһ„бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ, бһ“бһ·бһ„ Adobe Photoshop бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҸбҹӢбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҮбһ¶бһҠбһҫбһҳбҹ”

Computer Network (бһ”бһҺбҹ’бһҸбһ¶бһүбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ)

бһ”бһҺбҹ’бһҸбһ¶бһүбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ–бһёбһҡ бһ¬бһ…бҹ’бһҡбһҫбһ“бһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бҹ” бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ (Internet) бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбһ–бҹ’бһңбһҗбҹ’бһ„бҹғ бһӮбһәбһҮбһ¶бһ”бһҺбҹ’бһҸбһ¶бһүбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҠбҹҸбһ’бҹҶбһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһӣбһҫбһ–бһ·бһ—бһ–бһӣбҹ„бһҖбҹ”

Virus & Security (бһҳбҹҒбһҡбҹ„бһӮ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–)

бһҳбҹҒбһҡбҹ„бһӮбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ (Malware/Virus) бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ¶бһҖбҹ’бһҡбһҖбҹӢбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһӣбһҪбһ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһҜбһҖбһҹбһ¶бһҡ бһ¬бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҒбһјбһ…бҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһҳбҹ’бһ…бһ¶бһҸбҹӢбһҳбҹҒбһҡбҹ„бһӮ (Antivirus), бһҖбһ¶бһҡбһҳбһ·бһ“бһ…бһ»бһ…бһӣбһҫбһҸбҹҶбһҺбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһӣбҹҒбһҒбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҡбһ№бһ„бһҳбһ¶бҹҶ бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

I/O Ports (бһҡбһ“бҹ’бһ’бһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ)

бһҡбһ“бҹ’бһ’бһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ (Ports) бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһҮбһ¶бһ…бҹ’бһҡбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһҮбһјбһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹҒбһү бһ“бһ·бһ„бһ…бһјбһӣбһҡбһңбһ¶бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бҹ” бһҡбһ“бҹ’бһ’бһҠбҹӮбһӣбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҳбһ¶бһ“бһҠбһјбһ…бһҮбһ¶ USB (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Flash Drive, Mouse), HDMI (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бҹҒбһүбһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ…бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ) бһ“бһ·бһ„ Audio Jack (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҹбһҹбҹ’бһҸбһ¶бһ”бҹӢбһҸбҹ’бһҡбһ…бҹҖбһҖ)бҹ”

Computer Maintenance (бһҖбһ¶бһҡбһҗбҹӮбһ‘бһ¶бҹҶбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ)

бһҖбһ¶бһҡбһҗбҹӮбһ‘бһ¶бҹҶбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһІбҹ’бһҷбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ№бһ„бһҮбһҪбһҷбһІбҹ’бһҷбһңбһ¶бһҳбһ¶бһ“бһўбһ¶бһҷбһ»бһҖбһ¶бһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“бһҷбһјбһҡ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһӣбҹ’бһўбҹ” бһҖбһ¶бһҡбһҗбҹӮбһ‘бһ¶бҹҶбһҡбһҪбһҳбһҳбһ¶бһ“бҹ– бһҖбһ¶бһҡбһҹбһҳбҹ’бһўбһ¶бһҸбһ’бһјбһӣбһёбһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ бһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё (Update Software) бһ“бһ·бһ„бһҖбһ¶бһҡбһҳбһ·бһ“бһ‘бһ¶бһүбһҷбһҖбһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бһ–бһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ”

Cloud Storage (бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӣбһҫбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ)

Cloud Storage бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҷбһҫбһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡбһ“бҹ…бһӣбһҫ Server бһҸбһ¶бһҳбһҡбһҷбҹҲбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– Google Drive, OneDrive, Dropbox) бһҮбҹҶбһ“бһҪбһҹбһІбҹ’бһҷбһҖбһ¶бһҡбһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„ Flash Drive бһ¬ Hard Disk бһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһңбһ¶бһҮбһҪбһҷбһІбҹ’бһҷбһҷбһҫбһ„бһўбһ¶бһ…бһ”бһҫбһҖбһҜбһҖбһҹбһ¶бһҡбһ–бһёбһӮбҹ’бһҡбһ”бҹӢбһ‘бһёбһҖбһ“бҹ’бһӣбҹӮбһ„ бһӮбҹ’бһҡбһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһ–бҹҒбһӣбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҒбһјбһ…бҹ”

Web Browser (бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ)

Web Browser бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһјбһӣбһҳбһҫбһӣбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ“бһ¶бһ“бһ¶бһ“бҹ…бһӣбһҫбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ бһҠбһјбһ…бһҮбһ¶ Google Chrome, Safari, Firefox бһ¬ Microsoft Edgeбҹ” бһңбһ¶бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһҖбһјбһҠ HTML, CSS бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ— бһІбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһҷбһҫбһ„бһўбһ¶бһ…бһҳбһҫбһӣбһҷбһӣбҹӢбҹ”

Printer & Scanner (бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ– бһ“бһ·бһ„бһҹбҹ’бһҖбҹҒбһ“)

Printer (бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–) бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бҹҒбһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Output) бһҠбҹӮбһӣбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһҜбһҖбһҹбһ¶бһҡбһ‘бһ“бҹӢ (Soft copy) бһ–бһёбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһ‘бҹ…бһҮбһ¶бһҜбһҖбһҹбһ¶бһҡбһҡбһ№бһ„ (Hard copy) бһӣбһҫбһҖбҹ’бһҡбһҠбһ¶бһҹбҹ” бһ…бҹҶбһҺбҹӮбһҖ Scanner (бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹ’бһҖбҹҒбһ“) бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Input) бһҠбҹӮбһӣбһҗбһҸбһ…бһҳбҹ’бһӣбһ„бһҜбһҖбһҹбһ¶бһҡбһӣбһҫбһҖбҹ’бһҡбһҠбһ¶бһҹ бһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҡбһјбһ”бһ—бһ¶бһ– бһ¬бһҜбһҖбһҹбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһңбһ·бһүбҹ”

Troubleshooting (бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶)

Troubleshooting бһӮбһәбһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹғбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҠбһҫбһҡбһҳбһ·бһ“бһ”бҹ’бһҡбһҖбҹ’бһҡбһҸбһё бһ¬бһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹ (Error)бҹ” бһҮбҹҶбһ бһ¶бһ“бһҠбҹҶбһ”бһјбһ„ бһ“бһ·бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбһ¶бҹҶбһ„ бһ¬бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҮбһ¶бһ”бҹӢбһӮбһ¶бҹҶбһ„ бһӮбһәбһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ Restart (бһ”бһ·бһ‘бһ”бһҫбһҖбһҗбҹ’бһҳбһё) бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

Ergonomics (бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҹбһ»бһҒбһ»бһҳбһ¶бһӣбһ—бһ¶бһ–)

Ergonomics бһҹбҹҶбһҠбҹ…бһӣбһҫбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҖбһ“бҹ’бһӣбҹӮбһ„бһўбһ„бҹ’бһӮбһ»бһҷбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһІбҹ’бһҷбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһ”бһүбҹ’бһ бһ¶бһҹбһ»бһҒбһ—бһ¶бһ–бһҠбһјбһ…бһҮбһ¶ бһҲбһәбһҒбҹ’бһ“бһ„ бһ¬бһҹбҹ’бһҡбһңбһ¶бҹҶбһ„бһ—бҹ’бһ“бҹӮбһҖбҹ” бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһўбһ„бҹ’бһӮбһ»бһҷбһІбҹ’бһҷбһҸбҹ’бһҡбһ„бҹӢбһҒбҹ’бһ“бһ„ бһ‘бһ»бһҖбһӮбһҳбҹ’бһӣбһ¶бһҸбһ–бһёбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺбһҳбһҪбһҷбһ…бһ„бҹ’бһўбһ¶бһҳбһҠбҹғбһҖбһ“бҹ’бһӣбҹҮ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ…бҹ’бһ”бһ¶бһ”бҹӢ 20-20-20 (бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢ бҹўбҹ бһ“бһ¶бһ‘бһё бһҳбһҫбһӣбһ‘бҹ…бһҶбҹ’бһ„бһ¶бһҷбһ…бһҳбҹ’бһ„бһ¶бһҷ бҹўбҹ бһ бҹ’бһңбһёбһҸ бһҡбһҷбҹҲбһ–бҹҒбһӣ бҹўбҹ бһңбһ·бһ“бһ¶бһ‘бһё) бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбһ¶бһҖбһ—бҹ’бһ“бҹӮбһҖбҹ”

Cybersecurity (бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһўбһ»бһёбһ“бһ’бһәбһҺбһ·бһҸ)

бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһўбһ»бһёбһ“бһ’бһәбһҺбһ·бһҸ бһ¬ Cybersecurity бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһңбһ·бһ’бһ¶бһ“бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһ”бһҺбҹ’бһҸбһ¶бһү бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡ бһ¬бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹӮбһӣбһҡбһ№бһ„бһҳбһ¶бҹҶ бһҖбһ¶бһҡбһ”бһҫбһҖбһҳбһ»бһҒбһ„бһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбҹўбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ 2FA (Two-Factor Authentication) бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“бһ…бҹҶбһ–бҹ„бҹҮбһҹбһ¶бһҡбһ”бҹ„бһҖбһ”бҹ’бһҡбһ¶бһҹбҹӢ (Phishing) бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҒбҹ’бһӣбһҪбһ“бһҠбҹҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ”

Artificial Intelligence (бһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ)

бһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ (AI) бһҹбҹҶбһҠбҹ…бһӣбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ бһ¬бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһҠбҹӮбһӣбһўбһ¶бһ…бһӮбһ·бһҸ бһҡбҹҖбһ“бһҹбһјбһҸбҹ’бһҡ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ”бҹ’бһҡбһ бһ¶бһҖбҹӢбһ”бҹ’бһҡбһ бҹӮбһӣбһҳбһ“бһ»бһҹбҹ’бһҹбҹ” бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ AI бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһ—бһ¶бһҹбһ¶ бһҖбһ¶бһҡбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ•бҹ’бһ‘бҹғбһҳбһ»бһҒ бһҡбһҗбһҷбһ“бҹ’бһҸбһ”бһҫбһҖбһ”бһҡбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһ“бһ·бһ„бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһ“бһ·бһҳбҹ’бһҳбһ·бһҸ (Virtual Assistants) бһҠбһјбһ…бһҮбһ¶ ChatGPT бһҮбһ¶бһҠбһҫбһҳбҹ”

Programming Languages (бһ—бһ¶бһҹбһ¶бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ)

бһ—бһ¶бһҹбһ¶бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ бһӮбһәбһҮбһ¶бһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһҳбһ“бһ»бһҹбҹ’бһҹбһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҸбһӣбҹӢбһ”бһ‘бһ”бһүбҹ’бһҮбһ¶бһІбҹ’бһҷбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбҹ” бһ—бһ¶бһҹбһ¶бһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҡбҹҖбһ„бҹ—бһҒбҹ’бһӣбһҪбһ“ бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Python бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бһ·бһ„ AI, JavaScript бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһңбҹӮбһ”бһҹбһ¶бһҷ бһ“бһ·бһ„ C++ бһ¬ C# бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ бҹ’бһӮбҹҒбһҳбһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бҹҶбҹ—бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбҹ”

Data Backup (бһҖбһ¶бһҡбһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ)

бһҖбһ¶бһҡбһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Backup) бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҗбһҸбһ…бһҳбҹ’бһӣбһ„бһҜбһҖбһҹбһ¶бһҡ бһ¬бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ‘бһ»бһҖбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһҹбҹҒбһ„бһҳбһҪбһҷбһ‘бҹҖбһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ»бҹҶбһІбҹ’бһҷбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһ“бҹ…бһ–бҹҒбһӣбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶ бһҳбҹҒбһҡбҹ„бһӮбһ…бһјбһӣ бһ¬бһӣбһ»бһ”бһҠбҹ„бһҷбһўбһ…бҹҒбһҸбһ“бһ¶бҹ” бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ 3-2-1 бһҮбһ¶бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһӮбһә бһҗбһҸбһ…бһҳбҹ’бһӣбһ„ бҹЈ бһ…бҹ’бһ”бһ¶бһ”бҹӢ бһ‘бһ»бһҖбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ бҹў бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„ бҹЎ бһ…бҹ’бһ”бһ¶бһ”бҹӢбһ‘бһ»бһҖбһ“бҹ…бһҖбҹ’бһҡбҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ бһ¬бһ‘бһ»бһҖбһӣбһҫ Cloudбҹ”

Database (бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ)

бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Database) бһӮбһәбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶ бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһҹбһҺбҹ’бһҸбһ¶бһ”бҹӢбһ’бҹ’бһ“бһ¶бһ”бҹӢ бһҠбҹӮбһӣбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (DBMS) бһҠбһјбһ…бһҮбһ¶ MySQL, Oracle бһ¬ MongoDB бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ“бһ·бһ„бһңбҹӮбһ”бһҹбһ¶бһҷбһ“бһ¶бһ“бһ¶бһўбһ¶бһ…бһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹ бһ“бһ·бһ„бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ”

Web Development (бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһңбҹӮбһ”бһҹбһ¶бһҷ)

бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһңбҹӮбһ”бһҹбһ¶бһҷ (Web Development) бһӮбһәбһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹғбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹӮбһ„бһ…бҹӮбһҖбһҮбһ¶бһ–бһёбһҡбһ•бҹ’бһ“бҹӮбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһӮбһә Front-end (бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҳбһ»бһҒбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбһҫбһӣбһғбһҫбһү) бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ HTML, CSS бһ“бһ·бһ„ JavaScript бһ“бһ·бһ„ Back-end (бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ¬ Server) бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ—бһ¶бһҹбһ¶бһҠбһјбһ…бһҮбһ¶ PHP, Python бһ¬ Node.js бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

Mobile App Development (бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘)

бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘ бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҶбҹ’бһӣбһ¶бһҸбһңбҹғ бһҠбһјбһ…бһҮбһ¶ Android бһ“бһ·бһ„ iOSбҹ” бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ—бһ¶бһҹбһ¶бһҠбһҫбһҳ (Native) бһҠбһјбһ…бһҮбһ¶ Kotlin бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Android бһ“бһ·бһ„ Swift бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ iOS бһ¬бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ Cross-platform бһҠбһјбһ…бһҮбһ¶ Flutter бһ“бһ·бһ„ React Native бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҸбҹӮбһҳбҹ’бһҸбһ„ бһўбһ¶бһ…бһҠбһҫбһҡбһ”бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ”

UI/UX Design (бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ)

UI/UX Design бһӮбһәбһҮбһ¶бһҮбҹҶбһ“бһ¶бһүбһҡбһ…бһ“бһ¶бһҠбҹӮбһӣбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҹбҹ„бһ—бҹҗбһҺбһ—бһ¶бһ– бһ“бһ·бһ„бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ” UI (User Interface) бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ–бһҺбҹҢ бһўбһҖбҹ’бһҹбһҡ бһ“бһ·бһ„бһ”бҹҠбһјбһҸбһ»бһ„ бһ…бҹҶбһҺбҹӮбһҖ UX (User Experience) бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ‘бһјбһ‘бҹ…бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһҗбһ¶бһҸбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ бһ¬бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹ„бҹҮбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ…бһ»бһ… бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹӮбһҡбһ¬бһ‘бҹҒбҹ”

Cloud Computing (бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҖбҹ’бһӣбҹ„бһҠ)

бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҖбҹ’бһӣбҹ„бһҠ (Cloud Computing) бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһҠбһјбһ…бһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ (Servers) бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһҸбһ¶бһҳбһҡбһҷбҹҲбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбҹ” бһҮбҹҶбһ“бһҪбһҹбһІбҹ’бһҷбһҖбһ¶бһҡбһ‘бһ·бһүбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ Server бһҗбҹ’бһӣбҹғбҹ—бһ‘бһ»бһҖбһ“бҹ…бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҮбһҪбһӣбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ–бһёбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһ’бҹҶбҹ—бһҠбһјбһ…бһҮбһ¶ Amazon Web Services (AWS), Google Cloud бһ¬ Microsoft Azure бһҠбҹ„бһҷбһ…бҹҶбһҺбһ¶бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

Internet of Things (бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһ“бҹғбһңбһҸбҹ’бһҗбһ»)

бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһ“бҹғбһңбһҸбҹ’бһҗбһ» (IoT) бһҹбҹҶбһҠбҹ…бһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғ бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҳбһјбһӣ бһ“бһ·бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбһјбһ…бһҮбһ¶ бһўбҹҶбһ–бһјбһӣбһ—бҹ’бһӣбһҫбһ„бһҶбҹ’бһӣбһ¶бһҸбһңбҹғ бһ‘бһјбһҡбһ‘бһҹбҹ’бһҹбһ“бҹҚбһҶбҹ’бһӣбһ¶бһҸбһңбҹғ бһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһ“бһ¶бһЎбһ·бһҖбһ¶бһҶбҹ’бһӣбһ¶бһҸбһңбҹғ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҷбһҫбһ„бһ”бһүбҹ’бһҮбһ¶ бһ“бһ·бһ„бһҸбһ¶бһҳбһҠбһ¶бһ“бһ–бһҪбһҖбһңбһ¶бһ–бһёбһ…бһҳбҹ’бһ„бһ¶бһҷбһҸбһ¶бһҳбһҡбһҷбҹҲбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҠбҹғбҹ”

Git & GitHub (бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮбһҖбһјбһҠ)

бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮ (Version Control System) бһҠбһјбһ…бһҮбһ¶ Git бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹғбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбһјбһҠ бһ“бһ·бһ„бһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ…бһҡбһҖбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ…бһ¶бһҹбҹӢбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹбҹ” бһ…бҹҶбһҺбҹӮбһҖ GitHub бһӮбһәбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһ‘бһ»бһҖ (Hosting) бһӣбһҫбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҠбҹӮбһӣбһҮбһҪбһҷбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бһ¶бһҖбҹӢбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһӣбһҫбһӮбһҳбҹ’бһҡбҹ„бһ„бһҸбҹӮбһҳбһҪбһҷбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбҹ”

Data Science (бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ)

бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Data Science) бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһ‘бһ¶бһүбһҷбһҖбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһ¬бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹҸбһ…бҹ’бһҡбһҫбһ“бһҹбһ“бҹ’бһ’бһ№бһҖбһҹбһ“бҹ’бһ’бһ¶бһ”бҹӢ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶ бһҹбҹ’бһҗбһ·бһҸбһ· бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ (бһҠбһјбһ…бһҮбһ¶ Python бһ¬ R)бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһ“бһ·бһ“бҹ’бһ“бһ¶бһҖбһ¶бһҡбһ‘бһёбһ•бҹ’бһҹбһ¶бһҡ бһ“бһ·бһ„бһҮбһҪбһҷбһҠбһӣбҹӢбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҖбҹ’бһ“бһ»бһ„бһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбҹ”

Blockchain (бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ”бҹ’бһӣбһ»бһҖбһҶбҹҒбһ“)

Blockchain бһӮбһәбһҮбһ¶бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹӮбһ”бһңбһ·бһҳбһҮбҹ’бһҲбһҖбһ¶бһҡ (Decentralized) бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ бһ“бһ·бһ„бһҳбһ·бһ“бһўбһ¶бһ…бһӣбһҪбһ…бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһ¶бһ“бһЎбһҫбһҷбҹ” бһңбһ¶бһӮбһәбһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮбһ“бҹғбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҢбһёбһҮбһёбһҗбһӣ (Cryptocurrency) бһҠбһјбһ…бһҮбһ¶ Bitcoin бһ“бһ·бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһ бһ·бһҡбһүбҹ’бһүбһңбһҸбҹ’бһҗбһ» бһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһҷбһ¶бһҶбҹ’бһӣбһ¶бһҸбһңбҹғ (Smart Contracts) бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҒбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҶбҹ’бһӣбһ„бһ”бҹ’бһҡбһ‘бҹҒбһҹбҹ”

AR & VR (бһҖбһ¶бһҡбһ–бһ·бһҸбһ“бһ·бһҳбҹ’бһҳбһ·бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ·бһҸбһ–бһ„бҹ’бһҡбһёбһҖ)

VR (Virtual Reality) бһ”бһүбҹ’бһ…бһјбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһҸбһ¶бһҳбһҡбһҷбҹҲбһңбҹүбҹӮбһ“бһҸбһ¶бһ–бһ·бһҹбҹҒбһҹ бһ…бҹҶбһҺбҹӮбһҖ AR (Augmented Reality) бһ”бһүбҹ’бһ…бһјбһӣбһҡбһјбһ”бһ—бһ¶бһ–бһҢбһёбһҮбһёбһҗбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– бһ бҹ’бһӮбҹҒбһҳ PokГ©mon GO бһ¬бһҸбһҳбҹ’бһҡбһ„бһҳбһ»бһҒбһҖбҹ’бһ“бһ»бһ„ TikTok)бҹ” бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮбһҖбҹҶбһ–бһ»бһ„бһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһ бҹ’бһӮбҹҒбһҳ бһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһңбҹҒбһҮбҹ’бһҮбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбҹ”

Digital Marketing (бһ‘бһёбһ•бҹ’бһҹбһ¶бһҡбһҢбһёбһҮбһёбһҗбһӣ)

бһ‘бһёбһ•бҹ’бһҹбһ¶бһҡбһҢбһёбһҮбһёбһҗбһӣ бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ•бһӣбһ·бһҸбһ•бһӣ бһ¬бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҢбһёбһҮбһёбһҗбһӣ бһҠбһјбһ…бһҮбһ¶ Facebook, Google, бһ“бһ·бһ„ YouTubeбҹ” бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҖбһ¶бһҡбһ’бҹ’бһңбһҫ SEO (Search Engine Optimization), бһҖбһ¶бһҡбһҡбһҸбҹӢбһ–бһ¶бһҺбһ·бһҮбҹ’бһҮбһҖбһҳбҹ’бһҳ (Ads) бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶ (Content) бһӮбһәбһҮбһ¶бһҮбҹҶбһ“бһ¶бһүбһҠбҹҸбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһҹбһҳбҹҗбһҷбһҗбҹ’бһҳбһёбҹ”

Premiere Pro (бһҖбһ¶бһҸбҹӢбһҸбһңбһёбһҠбҹҒбһўбһјбһҖбһҳбҹ’бһҡбһ·бһҸбһўбһ¶бһҮбһёбһ–)

Adobe Premiere Pro бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһ¶бһҸбҹӢбһҸбһңбһёбһҠбҹҒбһўбһјбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһ—бһ¶бһ–бһҷбһ“бҹ’бһҸ бһ‘бһјбһҡбһ‘бһҹбҹ’бһҹбһ“бҹҚ бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһҹбһ„бҹ’бһӮбһҳбҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҖбһ¶бһҸбҹӢбһҸбһңбһёбһҠбҹҒбһўбһј бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҶбһЎбҹҒбһ„ бһҖбҹӮбһ–бһҺбҹҢ (Color Grading) бһ“бһ·бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбһҖбҹ’бһҹбһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮбҹ” Adobe Premiere Pro бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһ¶бһҸбҹӢбһҸбһңбһёбһҠбҹҒбһўбһјбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһ—бһ¶бһ–бһҷбһ“бҹ’бһҸ бһ‘бһјбһҡбһ‘бһҹбҹ’бһҹбһ“бҹҚ бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһҹбһ„бҹ’бһӮбһҳбҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҖбһ¶бһҸбҹӢбһҸбһңбһёбһҠбҹҒбһўбһј бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҶбһЎбҹҒбһ„ бһҖбҹӮбһ–бһҺбҹҢ (Color Grading) бһ“бһ·бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбһҖбҹ’бһҹбһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮбҹ”

After Effects (бһ…бһӣбһ“бһ¶бһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһҖбһ“бһ·бһ„бһ”бҹӮбһ”бһ•бҹӮбһ“)

бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһ¶бһҸбҹӢбһҸбһ‘бһјбһ‘бҹ… Adobe After Effects бһ•бҹ’бһҸбҹ„бһҸбһ‘бҹ…бһӣбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһӣбһ“бһ¶бһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһҖ (Motion Graphics) бһ“бһ·бһ„бһ”бҹӮбһ”бһ•бҹӮбһ“бһ–бһ·бһҹбҹҒбһҹ (Visual Effects - VFX) бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбһҖбҹ’бһҹбһҡбһҳбһ¶бһ“бһ…бһӣбһ“бһ¶ бһҖбһ¶бһҡбһӣбһ»бһ”бһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷбһ–бһҺбҹҢбһ”бҹғбһҸбһ„ (Green Screen) бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһӣбһ“бһ¶ 2D/3Dбҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһ¶бһҸбҹӢбһҸбһ‘бһјбһ‘бҹ… Adobe After Effects бһ•бҹ’бһҸбҹ„бһҸбһ‘бҹ…бһӣбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһӣбһ“бһ¶бһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһҖ (Motion Graphics) бһ“бһ·бһ„бһ”бҹӮбһ”бһ•бҹӮбһ“бһ–бһ·бһҹбҹҒбһҹ (Visual Effects - VFX) бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбһҖбҹ’бһҹбһҡбһҳбһ¶бһ“бһ…бһӣбһ“бһ¶ бһҖбһ¶бһҡбһӣбһ»бһ”бһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷбһ–бһҺбҹҢбһ”бҹғбһҸбһ„ (Green Screen) бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһӣбһ“бһ¶ 2D/3Dбҹ”

Mobile Editing (бһҖбһ¶бһҸбҹӢбһҸбһңбһёбһҠбҹҒбһўбһјбһӣбһҫбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘)

бһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһҸбһңбһёбһҠбҹҒбһўбһјбһ“бҹ…бһӣбһҫбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҠбҹғбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбһјбһ…бһҮбһ¶ CapCut бһҖбҹҶбһ–бһ»бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶ (Content Creators) бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһңбһ¶бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫ бһҳбһ¶бһ“бһ”бҹӮбһ”бһ•бҹӮбһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ—бһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бһ¶бһ“бһҡбһ бҹҗбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮбһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһҹбһ„бҹ’бһӮбһҳбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһҸбһңбһёбһҠбҹҒбһўбһјбһ“бҹ…бһӣбһҫбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҠбҹғбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбһјбһ…бһҮбһ¶ CapCut бһҖбҹҶбһ–бһ»бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶ (Content Creators) бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһңбһ¶бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫ бһҳбһ¶бһ“бһ”бҹӮбһ”бһ•бҹӮбһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ—бһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бһ¶бһ“бһҡбһ бҹҗбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮбһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһҹбһ„бҹ’бһӮбһҳбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

Python Programming (бһ—бһ¶бһҹбһ¶ Python)

Python бһӮбһәбһҮбһ¶бһ—бһ¶бһҹбһ¶бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҠбҹӮбһӣбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһ”бҹҶбһ•бһ»бһҸбһҳбһҪбһҷбһ“бҹ…бһӣбһҫбһ–бһ·бһ—бһ–бһӣбҹ„бһҖ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһңбһ¶бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҡбҹҖбһ“ бһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҖбһјбһҠбһҒбҹ’бһӣбһё бһ“бһ·бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҒбһҷбһҖбһ‘бҹ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһңбҹӮбһ”бһҹбһ¶бһҷ бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Data Analysis) бһ“бһ·бһ„бһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ (AI)бҹ” Python бһӮбһәбһҮбһ¶бһ—бһ¶бһҹбһ¶бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҠбҹӮбһӣбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһ”бҹҶбһ•бһ»бһҸбһҳбһҪбһҷбһ“бҹ…бһӣбһҫбһ–бһ·бһ—бһ–бһӣбҹ„бһҖ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһңбһ¶бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҡбҹҖбһ“ бһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҖбһјбһҠбһҒбҹ’бһӣбһё бһ“бһ·бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҒбһҷбһҖбһ‘бҹ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһңбҹӮбһ”бһҹбһ¶бһҷ бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Data Analysis) бһ“бһ·бһ„бһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ (AI)бҹ”

Object-Oriented (бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹӮбһ”бһңбһҸбҹ’бһҗбһ»)

Object-Oriented Programming (OOP) бһӮбһәбһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ“бҹғбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҠбҹӮбһӣбһҡбҹҖбһ”бһ…бҹҶбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ "бһңбһҸбҹ’бһҗбһ»" (Objects) бһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбҹ” бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҡбһ”бһҹбҹӢбһңбһ¶бһҮбһҪбһҷбһұбҹ’бһҷбһҖбһјбһҠбһҳбһ¶бһ“бһҹбһҺбҹ’бһҸбһ¶бһ”бҹӢбһ’бҹ’бһ“бһ¶бһ”бҹӢ бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹӮбһ”бҹ’бһҡбҹӮ бһ“бһ·бһ„бһўбһ¶бһ…бһҷбһҖбһ‘бҹ…бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһүбһ”бһ¶бһ“бҹ” Object-Oriented Programming (OOP) бһӮбһәбһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ“бҹғбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҠбҹӮбһӣбһҡбҹҖбһ”бһ…бҹҶбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ "бһңбһҸбҹ’бһҗбһ»" (Objects) бһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбҹ” бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҡбһ”бһҹбҹӢбһңбһ¶бһҮбһҪбһҷбһұбҹ’бһҷбһҖбһјбһҠбһҳбһ¶бһ“бһҹбһҺбҹ’бһҸбһ¶бһ”бҹӢбһ’бҹ’бһ“бһ¶бһ”бҹӢ бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹӮбһ”бҹ’бһҡбҹӮ бһ“бһ·бһ„бһўбһ¶бһ…бһҷбһҖбһ‘бҹ…бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһүбһ”бһ¶бһ“бҹ”

Algorithms (бһҖбҹ’бһ”бһҪбһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’)

бһҖбҹ’бһ”бһҪбһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ (Algorithm) бһӮбһәбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ бһ…бҹҶбһҺбҹӮбһҖбһҜбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Data Structure) бһӮбһәбһҮбһ¶бһҡбһ”бҹҖбһ”бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһұбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһ…бҹҶбһҺбһ»бһ…бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮ бһӮбһәбһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮбһҠбҹҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ” бһҖбҹ’бһ”бһҪбһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ (Algorithm) бһӮбһәбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ бһ…бҹҶбһҺбҹӮбһҖбһҜбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Data Structure) бһӮбһәбһҮбһ¶бһҡбһ”бҹҖбһ”бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһұбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһ…бҹҶбһҺбһ»бһ…бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮ бһӮбһәбһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮбһҠбҹҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ”
